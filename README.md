# نظام إدارة الصيدلية - الدفاتر والإجراءات الحكومية

نظام ويب متكامل لإدارة الصيدليات وفقاً للدفاتر والإجراءات الموصوفة في المستند:
- اليومية (ت ص 9) مع نظام المراجعة والتعديل بالقلم الأحمر.
- دفتر 223 (تجميع المنصرف اليومي).
- دفتر 118 (دفتر العهدة الكبير).
- إذن 111 (طلبيات للمخزن) بخمس نسخ.
- دفتر 112 (الطلبيات الواردة) مع تسجيل تواريخ الصلاحية وأرقام التشغيلات.
- دفتر 5 (القيم المالية).
- نظام الجرد الدوري.
- تقارير شهرية حسب الشهر الحكومي (من 26 إلى 25).

## التقنيات المستخدمة
- Python 3.8+ مع Flask
- SQLite (قاعدة بيانات خفيفة، يمكن تبديلها)
- HTML5، CSS3، JavaScript، Bootstrap 5

## التثبيت والتشغيل
1. استنساخ المستودع:
git clone https://github.com/elkotitahmed/pharmacy-management-system.git
cd pharmacy-management-system
2. إنشاء بيئة افتراضية (اختياري):
python -m venv venv
source venv/bin/activate # Linux/Mac
venv\Scripts\activate # Windows
3. تثبيت المتطلبات:
pip install -r requirements.txt

4. تشغيل التطبيق:
python app.py

5. فتح المتصفح على: http://127.0.0.1:5000

## الهيكل
- `app.py`: الخادم الرئيسي وجميع الـ APIs.
- `models.py`: تعريف قاعدة البيانات.
- `templates/`: قوالب HTML.
- `static/`: ملفات CSS و JavaScript.

## الميزات الرئيسية
- إدارة الأصناف (إضافة، تعديل، حذف).
- تسجيل التذاكر اليومية مع إمكانية مراجعة الشطب وتعديل الكميات.
- تجميع تلقائي للمنصرف في دفتر 223.
- عرض دفتر العهدة 118 مع الوارد والمنصرف والرصيد.
- إنشاء أذون طلب 111 واستلامها 112 مع تحديث المخزون.
- دفتر القيم المالية 5.
- جرد الصيدلية وتسجيل الفروق.
- تقارير شهرية حسب التقويم الحكومي.

## إعدادات إضافية
- يمكن تغيير يوم بداية الشهر الحكومي من جدول `system_config`.
- يمكن رفع النظام على PythonAnywhere أو Render بسهولة.
